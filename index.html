About BIMI SVG Requirements

BIMI (Brand Indicators for Message Identification) requires SVG logos to follow specific standards:

  • Must be a valid SVG Tiny 1.2 format
  • Must not contain external references
  • Cannot include JavaScript or animation
  • Must be properly sized with viewBox attribute
  • Should have a transparent background

This validator helps ensure your logo meets these requirements for successful BIMI implementation.

Complete standard details can be found in the SVG Tiny 1.2 specification.

This tool is provided for free to help email marketers implement BIMI effectively.
